# CardioMax Booking System

This is the full CardioMax Fitness booking system scaffold with:
- Next.js frontend
- Stripe Checkout payments
- Supabase bookings storage
- SendGrid emails
- Twilio SMS
- Google Calendar integration
- Admin dashboard
- Reminders (cron)
- Cancel / Reschedule flows

Fill `.env` from `.env.example`, create the Supabase table, and run `npm install` then `npm run dev`.

See the canvas/notes for details.
